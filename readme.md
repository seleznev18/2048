# 2048 Game

Проект представляет собой веб-версию популярной игры 2048. Пользователь может передвигать плитки на игровом поле, объединять одинаковые числа, сохранять рекорды, отменять ход и продолжать игру с последнего состояния. Реализовано на чистом HTML, CSS и JavaScript без использования фреймворков. Для анимации движения и слияния плиток используется `animations.css`.

---

## Структура проекта

### `index.html`

Главная страница приложения. Содержит:

* контейнер игрового поля;
* блок текущего счета;
* модальное окно с таблицей лидеров;
* кнопки управления (для мобильных устройств).

### `css/style.css` и `css/animations.css`

* `style.css` — оформление игрового поля и плиток, адаптивная вёрстка для мобильных и десктопных устройств, цвета плиток соответствуют их значениям.  
* `animations.css` — плавная анимация движения и слияния плиток.

### `js/board.js`

* создание и рендер сетки 4x4;
* добавление новых плиток на пустые клетки;
* управление состоянием игрового поля.

### `js/game.js`

* логика движения плиток во всех направлениях;
* объединение плиток с начислением очков;
* проверка окончания игры;
* обновление счета.

### `js/ui.js`

* работа с DOM;
* отображение игрового поля и плиток;
* показ модального окна с рекордами;
* управление кнопками клавиатуры и виртуальными кнопками на мобильных устройствах;
* обработка кнопок «Начать заново» и «Отмена хода».

### `js/storage.js`

* сохранение состояния игры и рекордов в `localStorage`;
* восстановление состояния игры и топ-10 рекордов при повторном открытии;
* обновление данных таблицы рекордов.

### `js/main.js`

* инициализация игры;
* связывание логики игры, UI и хранения данных;
* управление глобальными событиями страницы.

---

## Реализованный функционал

* Динамическая генерация сетки 4x4 и начальных плиток;
* Движение плиток вверх, вниз, влево и вправо;
* Объединение плиток с начислением очков;
* Появление новых плиток в случайных пустых клетках;
* Проверка окончания игры;
* Сохранение и восстановление состояния игры и рекордов через `localStorage`;
* Кнопка «Начать заново» сбрасывает поле и счет без перезагрузки;
* Кнопка «Отмена хода» возвращает предыдущее состояние;
* Управление через клавиатуру на десктопе и виртуальные кнопки или свайпы на мобильных устройствах;
* Адаптивный интерфейс для разных экранов;
* Каждое значение плитки имеет уникальный цвет;
* Плавная анимация движения и слияния плиток с помощью `animations.css`.
